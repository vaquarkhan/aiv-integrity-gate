# Gate: `doc-integrity`

## Why it exists

Validates **documentation** in the diff: links resolve, referenced paths exist, and optional rules (commands, cross-refs) hold.

## When to enable

Use `--include-doc-checks` or enable the gate in `.aiv/config.yaml` when docs matter for your team.

## How to fix

- Correct broken links and paths; align examples with real commands in the repo.
